# ComfortSync

Minimal full-stack starter with:

- `frontend/`: React + Vite
- `backend/`: FastAPI

## Run

Frontend:

```powershell
cd frontend
& "C:\Program Files\nodejs\node.exe" "C:\Program Files\nodejs\node_modules\npm\bin\npm-cli.js" run dev
```

Backend:

```powershell
cd backend
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload
```
